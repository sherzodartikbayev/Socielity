# Socielity

**Socielity** is a social / community web application focused on connecting people in meaningful ways. It enables users to create profiles, share posts, and discover others based on shared interests. Built with a modern, responsive UI, Socielity makes it easy to engage, interact, and build relationships online.

## Key Features

- **User Profiles**: Sign up, set up a profile, and connect with others.  
- **Post Feed**: Share posts, see activity from your network.

## Motivation

In today’s digital world, people often want more than just vertical “like”-based networks. Socielity aims to foster **genuine connections** — not just for sharing content, but for meaningful interaction. Whether it’s finding friends around a hobby, starting discussions, or building a micro-community, Socielity empowers users to create and belong.
